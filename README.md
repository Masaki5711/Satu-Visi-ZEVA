# Satu Visi ZEVA — Official Anthem

YMMA / ZEVA をテーマにした楽曲「**Satu Visi ZEVA**」の公式配信サイトです。

## 内容
- 🎵 Full バージョン試聴 & ダウンロード
- 🎤 カラオケ (Instrumental) バージョン試聴 & ダウンロード
- 📜 歌詞表示
- 🔥 ライブ・カラオケモード（単語ごとに同期して光る）

## ファイル
| | |
|---|---|
| `index.html` | 公開サイト本体 |
| `sync.html` | 歌詞タイミング記録ツール（開発用） |
| `Satu Visi ZEVA (Full).mp3` | 楽曲本体 |
| `Satu Visi ZEVA(Instrumental).mp3` | カラオケ版 |
| `Satu Visi ZEVA（歌詞）.txt` | 歌詞テキスト |

## GitHub Pages の有効化

1. このリポジトリの **Settings → Pages**
2. **Source** を `Deploy from a branch` に
3. **Branch** を `main` / `/ (root)` に設定して保存

数十秒後に `https://masaki5711.github.io/Satu-Visi-ZEVA/` で公開されます。
